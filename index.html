School Organisation Design Dashboard 2026

School of Architecture & Built Environment
This dashboard is for creating the new organisational structure of the SoABE, based on the newly announced leadership positions. To update the org chart: create a discipline, then drag & drop the programs that should fall within it. You can add Course Coordinators to each discipline, and allocate a service % for every role - totals are calculated automatically. Scenarios let you save alternative structures and compare them side by side in the Scenario Comparison tab. In the Task Allocation tab you'll find the current list of Course Director tasks, ready to be allocated to each leadership role.
